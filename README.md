# Проект аквариума в Паттайе, Таиланд

## Общая информация

Проект создания и обслуживания пресноводного аквариума в условиях тропического климата Таиланда.

## Местоположение

- **Город:** Паттайя, Таиланд
- **Расположение:** Открытый балкон
- **Ориентация:** Лицевое стекло - ЮГ, заднее - СЕВЕР

## Цель проекта

Создание стабильной экосистемы аквариума с миролюбивыми видами рыб и растений, адаптированных к местным условиям.

## Особенности проекта

- Аквариум на открытом воздухе
- Тропический климат с высокой влажностью
- Использование местной водопроводной воды
- Минимальное использование дополнительного CO2

## Структура проекта

- `aquarium.md` - Основной файл с параметрами аквариума
- `fishes.md` - Подбор рыб (будет создан позже)
- `plants.md` - Подбор растений (будет создан позже)
- `equipment.md` - Оборудование и технические решения
- `maintenance.md` - График обслуживания

## Автор

Аквариумист-новичок, изучающий аквариумистику в условиях Таиланда.
